**Status:** ✅ SHIPPED 2026-06-29
**Phases:** 1-4
**Total Plans:** 14

## Overview

Build a UEFA Champions League competition module for the existing football prediction engine. The UCL module introduces the 2025+ Swiss-system format (36-team league phase, 8 matches per team, pot-constrained opponents), a two-legged knockout playoff (positions 9–24), a seeded R16 bracket with top-4 protection, and full Monte Carlo simulation — all without modifying `football_core`.
