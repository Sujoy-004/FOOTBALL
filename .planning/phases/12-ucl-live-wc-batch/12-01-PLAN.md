  Create `competitions/ucl/src/live_state.py` following the `worldcup/src/state.py` wrapper pattern:

  1. **Imports:** Import from `football_core.state`: `load_played`, `save_played`, `load_prediction_history`, `save_prediction_history`, `load_signal_cache`, `save_signal_cache`, `is_cache_valid`, `_atomic_write_json`. Also `json`, `os`, `pathlib.Path`, `typing`.

  2. **Data directory:** Hardcode `LIVE_DATA_DIR` as `os.path.join(os.path.dirname(os.path.abspath(__file__)), "..", "data", "live")` — resolved relative to `live_state.py` location.

  3. **`_resolve_live_dir(data_dir: str | None = None) -> str`:** Returns `data_dir` if provided, else `LIVE_DATA_DIR`. Creates directory if not exists (call `os.makedirs(exist_ok=True)`).

  4. **`load_ucl_played(data_dir=None) -> dict[str, dict]`:** Loads `ucl_played.json` from live dir. Returns `{}` on missing file (not an error). On `json.JSONDecodeError`, log warning and return `{}`.

  5. **`save_ucl_played(played: dict[str, dict], data_dir=None) -> None`:** Saves to `ucl_played.json` via `_atomic_write_json`.

  6. **`load_ucl_elo_applied(data_dir=None) -> list[str]`:** Loads `ucl_elo_applied.json` from live dir. Returns `[]` on missing file. This is a list of BSD match_ids that had Elo applied (restart guard per D-03).

  7. **`save_ucl_elo_applied(elo_applied: list[str], data_dir=None) -> None`:** Saves to `ucl_elo_applied.json` via `_atomic_write_json`.

  8. **`load_ucl_prediction_history(data_dir=None) -> list[dict]`:** Loads `ucl_prediction_history.json`. Returns `[]` on missing.

  9. **`save_ucl_prediction_history(history: list[dict], data_dir=None) -> None`:** Saves to `ucl_prediction_history.json` via `_atomic_write_json`.

  10. **`append_ucl_prediction_history(entry: dict, data_dir=None) -> None`:** Loads history, appends entry, saves back via `_atomic_write_json`. Atomic read-modify-write.

  11. **`load_ucl_cache(cache_key: str, data_dir=None) -> dict | None`:** Delegates to `football_core.state.load_signal_cache` with `data_dir=_resolve_live_dir(data_dir)`.

  12. **`save_ucl_cache(cache_key: str, data: dict, data_dir=None, ttl_hours=12) -> None`:** Delegates to `football_core.state.save_signal_cache`.

  13. **`ucl_is_cache_valid(cache, ttl_hours=12) -> bool`:** Delegates to `football_core.state.is_cache_valid`.

  **Key constraints (per D-01):**
  - All files use `ucl_` prefix: `ucl_played.json`, `ucl_elo_applied.json`, `ucl_prediction_history.json`
  - All writes use atomic pattern via existing `_atomic_write_json` — never `open()` + `write()` directly
  - Missing files always return empty default (never raise FileNotFoundError)
  - Corrupted JSON files log warning via `logging.warning` and return empty default
  - Directory `competitions/ucl/data/live/` is created if missing

  **Module docstring:** `"""UCL Live Monitor state persistence — thin wrapper around football_core.state for UCL-specific paths and file names."""`

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| in-memory → filesystem | State data crosses from Python process to JSON files on disk |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-12-01 | Tampering | Atomic write race condition | mitigate | Use `tempfile.mkstemp` + `os.replace` (atomic on same filesystem) via `_atomic_write_json` — never direct `open()` + `write()` |
| T-12-02 | Tampering | Stale state corruption on crash | mitigate | Signal handler only sets `running=False`; state save happens in normal loop flow after successful data processing |
| T-12-03 | Spoofing | Missing/corrupted state files | mitigate | All load functions return empty defaults on missing or `json.JSONDecodeError`; never raise FileNotFoundError |
</threat_model>
