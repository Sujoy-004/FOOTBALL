Fixture JSON schema (from RESEARCH.md §Fixture Schedule Data Model):
- Top-level: `{"schedule": {"teams": [...], "matchdays": [...]}}`
- teams entry: `{"name": str, "pot": int, "clubelo_name": str, "coefficient": float}`
- matchdays: array of 8 arrays, each containing 18 match objects
- match: `{"match_id": str, "team_a": str, "team_b": str, "home_pot": int, "away_pot": int}`

Validation constraints (UCLT-00):
1. Exactly 36 teams in `schedule.teams`
2. Each team appears exactly 8 times (4 home + 4 away) in matchups
3. Each team faces exactly 2 opponents from each pot
4. No duplicate matchups — each pair appears at most once
5. 8 matchdays, each with exactly 18 matches
6. All team names in matchups reference valid entries in `schedule.teams`
