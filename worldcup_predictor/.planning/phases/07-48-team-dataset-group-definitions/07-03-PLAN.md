    1. Verify `groups` param is a dict with key `"groups"` — raise `ValueError("groups data must contain 'groups' key")` if missing
    2. Verify `groups["groups"]` has exactly `constants.GROUP_COUNT` keys — raise `ValueError(f"Expected {constants.GROUP_COUNT} groups, got {len(actual)}")`
    3. Verify group keys are exactly `set("ABCDEFGHIJKL")` — raise `ValueError` with mismatched keys if not
    4. For each group (iterate `sorted(group_keys)` for deterministic order):
       a. Verify both `"teams"` and `"matches"` keys exist — raise `ValueError(f"Group {letter}: missing 'teams' or 'matches' key")`
       b. Verify `"teams"` has exactly `constants.TEAMS_PER_GROUP` entries — raise `ValueError(f"Group {letter}: expected {constants.TEAMS_PER_GROUP} teams, got {len(actual)}")`
       c. Verify `"matches"` has exactly `constants.MATCHES_PER_GROUP` entries — raise `ValueError(f"Group {letter}: expected {constants.MATCHES_PER_GROUP} matches, got {len(actual)}")`
       d. Verify each team name in `"teams"` is a non-empty string
       e. For each match in `"matches"`: verify it has `match_id`, `team_a`, `team_b` keys; verify `winner`, `score_a`, `score_b` are present (values can be None)
       f. Verify each match_id starts with `"GS_{letter}_"` (uppercase group letter) — raise `ValueError(f"Group {letter}: match_id '{mid}' does not start with 'GS_{letter}_'")`
    5. Verify no team appears in more than one group (use a `seen_teams: set[str]`, track across all groups) — raise `ValueError(f"Team '{team}' appears in multiple groups")`
    6. Verify all match_ids are unique across all groups — raise `ValueError(f"Duplicate match_id across groups: {mid}")`
    7. If `teams` argument is provided and is not None: verify every team name referenced in any group's `"teams"` array is a key in the `teams` dict — raise `ValueError(f"Team '{team}' not found in teams data")`

    Return type: `None`. Raises `ValueError` with first failure.
