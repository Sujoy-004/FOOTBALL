    Create `src/output.py` implementing the exact output blocks from UI-SPEC.md §6.

    **Module structure (top to bottom):**

    1. **Imports:** `sys`, `time`, `Callable` from typing

    2. **Private ANSI system:**
       - `_supports_color() -> bool` — returns `sys.stdout.isatty()`
       - `_ansi(code: str) -> Callable[[str], str]` — factory returns wrapper that applies `\033[{code}m{text}\033[0m` when color supported, else returns text as-is
       - Color wrappers: `_dim = _ansi("2")`, `_bold_cyan = _ansi("1;36")`, `_green = _ansi("32")`, `_red = _ansi("31")`, `_bold_green = _ansi("1;32")`, `_bold_white = _ansi("1;37")`, `_bold_yellow = _ansi("1;33")`, `_bold_red = _ansi("1;31")`
       - `_timestamp() -> str` — returns `_dim(f"[{time.strftime('%Y-%m-%d %H:%M:%S')}]")`

    3. **`print_probability_table(probs, prev_probs=None)`:**
       - Sort teams by `probs[name]["champion"]` descending
       - Extract top 5, remaining
       - Print timestamp line with label: "Initial probabilities:" if first call, "UPDATED PROBABILITIES:" otherwise
       - Print bold cyan header: `f"{'':>3} {'Team':<18} {'QF':>6} {'SF':>6} {'FINAL':>8} {'CHAMPION':>8}"` + optional "  {'Delta':>8}"
       - Print dash separator: `_bold_cyan("-" * (51 + (9 if prev_probs else 0)))`
       - For each top-5 team, print rank, name, qf/sf/final/champion values with `:.3f`
       - If prev_probs: compute `delta = probs[name]["champion"] - prev_probs[name]["champion"]`, choose ▲/▼, color green/red, format `{symbol} {delta:+.3f}`, right-align in 8-wide field
       - Print remaining teams one-liner: `f" ─── {len(remaining)} other teams — best: {best_name} ({best_val:.3f})"` — plain text, no color
       - Trailing blank line `print()`

    4. **`print_delta_summary(probs, prev_probs)`:**
       - Compute `(name, champion_delta)` for all teams
       - Sort by delta descending → risers (top 3), fallers (bottom 3, reversed for most-negative-first)
       - Print bold cyan "Biggest Risers" header
       - For each riser: `f"  {_green(f'▲ {name:<16} {delta:+.1%}')}"`
       - Blank line
       - Print bold cyan "Biggest Fallers" header
       - For each faller: `f"  {_red(f'▼ {name:<16} {delta:+.1%}')}"`
       - Trailing blank line

    5. **`print_simulation_duration(elapsed_seconds: float):`**
       - Print: `_bold_green(f"Re-simulating (50000 runs)... done in {elapsed_seconds:.1f}s")` — with timestamp prefix from `_timestamp()`

    **Key formatting rules from UI-SPEC §4 Color Rules:**
    - Timestamps ALWAYS dim gray — never change color
    - Delta ▲/▼ and the value share the same color (both green or both red)
    - Header row and dash separator are bold cyan
    - Inline deltas: `:.3f` decimal format
    - Risers/fallers deltas: `:.1%` percentage format

    **Write all functions as pure functions** (no classes, no module-level state). Color detection happens per-call via `_supports_color()`.
