  <action>
  Create `src/groups.py` implementing the following functions per the ARCHITECTURE.md §5.2 and CONTEXT.md D-01–D-07, D-09–D-12, D-16 specifications.

  **Function 1: `expected_goals(rating_a: float, rating_b: float, base_rate: float | None = None) -> float`**
  - Returns `adj_base * (10 ** ((rating_a - rating_b) / 400))` — the Elo-to-goals formula
  - `adj_base` = `(base_rate or constants.EXPECTED_GOALS_BASE_RATE) * 1.05` for team_a (home advantage, per D-03 and SPECIFICS section)
  - Imports `src.constants` and `src.elo.expected_score` is NOT used here — this function is a different formula (goal rate, not win probability). Use `10^((Δelo)/400)` factor applied to base_rate.
  - The `expected_goals` return value is the Poisson lambda parameter for team A's expected goals against team B.

  **Function 2: `_poisson_sample(lam: float, rng: random.Random) -> int`**
  - Per D-01: Implements the Knuth algorithm:
    - k = 0; p = 1.0; bound = math.exp(-lam)
    - while p > bound: k += 1; p *= rng.random()
    - return k - 1
  - Uses `rng.random()` (the passed-in Random instance), not module-level random

  **Function 3: `_simulate_single_match(team_a: str, team_b: str, elo_a: float, elo_b: float, rng: random.Random) -> dict`**
  - Returns dict with `{team_a, team_b, score_a, score_b, winner, yellow_cards_a, red_cards_a, yellow_cards_b, red_cards_b}`
  - Computes lambda_a via `expected_goals(elo_a, elo_b)` and lambda_b via `expected_goals(elo_b, elo_a)`
  - Samples scores via `_poisson_sample(lambda_a, rng)` and `_poisson_sample(lambda_b, rng)`
  - Winner logic: if score_a > score_b → team_a; if score_b > score_a → team_b; else None (draw)
  - Card draw per D-04: per team, YC = Poisson(2.0), RC = Poisson(0.05)
  - Per D-07 simplicity: conduct deduction not computed here; raw card counts stored

  **Function 4: `simulate_group_matches(groups: dict, teams: dict[str, dict], elo_ratings: dict[str, float], rng: random.Random) -> dict[str, dict[str, dict]]`**
  - Signature per ARCHITECTURE §5.2 (without `played` param — deferred to Phase 10)
  - Iterates all 12 groups → for each match in group["matches"]:
    - Looks up Elo from `elo_ratings[match["team_a"]]` and `elo_ratings[match["team_b"]]`
    - Calls `_simulate_single_match(team_a, team_b, elo_a, elo_b, rng)`
    - Sets `match["winner"]`, `match["score_a"]`, `match["score_b"]` in the results dict
  - Returns `{group_letter: {match_id: {team_a, team_b, score_a, score_b, winner, yellow_cards_a, red_cards_a, yellow_cards_b, red_cards_b}}}`
  - Does NOT mutate the input groups dict (operates on a per-iteration fresh results dict, per D-11)
  - Does NOT write to `winner_progression` (per D-12 — that starts at R32 in Phase 9)
