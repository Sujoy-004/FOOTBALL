  **Structure (follow existing patterns):**
  ```python
  #!/usr/bin/env python3
  """Benchmark for Phase 8 Group Stage Simulation Engine.

  Measures time for the full group pipeline at 1K, 10K, and 50K iterations.
  Results saved to BENCHMARK_RESULTS_08.md for cross-Phase 9 comparison.
  """
  ```

  **Data loading (once, outside the timed loop):**
  - Load `groups.json`, `teams.json`, `annex_c.json` from `constants.DATA_DIR` using `json.load()`
  - Build `elo_ratings` dict from teams data
  - Create a `random.Random(0)` instance (fixed seed for reproducibility)

  **Benchmark function: `benchmark_pipeline(iterations: int, groups, teams, elo_ratings, annex_c, rng_seed: int = 0) -> float`:**
  1. Create a new `random.Random(rng_seed)` instance for this run
  2. Record start time with `time.perf_counter()`
  3. Loop for `iterations`:
     - `results = simulate_group_matches(groups, teams, elo_ratings, rng)`
     - `standings = compute_standings(results)`
     - `third_ranked = rank_third_placed(standings)`
     - `advancers = select_advancers(standings, third_ranked)`
     - `r32 = resolve_r32_matchups(advancers, standings, third_ranked, annex_c)`
  4. Record end time, return elapsed seconds

  **Main block:**
  ```python
  if __name__ == "__main__":
      import time, json, random
      from src import constants
      from src.groups import simulate_group_matches, compute_standings, rank_third_placed, select_advancers, resolve_r32_matchups
      
      # Load data
      groups = json.load(open(constants.DATA_DIR / "groups.json"))
      teams = json.load(open(constants.DATA_DIR / "teams.json"))
      annex_c = json.load(open(constants.DATA_DIR / "annex_c.json"))
      elo = {n: v["elo"] for n, v in teams.items()}
      
      # Run benchmarks
      for iters in [1000, 10000, 50000]:
          elapsed = benchmark_pipeline(iters, groups, teams, elo, annex_c)
          ops_per_sec = (iters * 72) / elapsed  # 72 match simulations per iteration
          print(f"{iters:>6} iterations: {elapsed:.3f}s  ({ops_per_sec:.0f} matches/s)")
  ```

  2. **Local variable bindings:** In `simulate_group_matches`, bind `expected_goals` and `_poisson_sample` to local variables outside the match loop to reduce attribute lookup overhead
  3. Report before/after times

  Add the `@lru_cache` import guard — only apply if the function signature is hashable (it is: all float parameters).
