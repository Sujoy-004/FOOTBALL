    1. `_deduplicate_history(entries: list[dict]) -> list[dict]`

       - Group by match_id, take last entry per match_id (list is chronologically ordered).
       - Return deduplicated list preserving chronological order.

    2. `_per_match_briers(entries: list[dict], signal_key: str) -> list[float]`
       - Per D-09, extracts per-match Brier scores for sigma computation.
       - For each entry: check `entry["signals"][signal_key]["available"]` = true, get probability + actual, compute `(p - a) ** 2`.
       - Returns ordered list of per-match Brier scores.
       - Same filtering logic as `blender.compute_rolling_brier()` but returns list, not mean.
       - Use `brier_score` from evaluation.py.

    3. `check_drift(entries: list[dict], signal_key: str, reference_baseline: float, window: int = 50, sigma_threshold: float = 2.0) -> dict | None`
       - Implements D-09 formula:
         ```
         rolling_sigma = std(per_match_briers in last window matches)
         drift_alert = rolling_mean > reference_baseline + 2 * rolling_sigma
         ```
       - If len(per_match_briers) < COLD_START_THRESHOLD (30): return None
       - rolling = per_match_briers[-window:] (last window)
       - rolling_mean = sum(rolling) / len(rolling)
       - If len(rolling) >= 2: compute population std via `math.sqrt(sum((x - mean)**2 for x in rolling) / len(rolling))`
       - Compare: rolling_mean > reference_baseline + sigma_threshold * sigma
       - If drifted: also check by how much (delta = rolling_mean - threshold)
       - Return dict with: signal, rolling_mean, reference_baseline, sigma, threshold, drifted (bool)

    4. `compute_reference_baselines(entries: list[dict], signal_keys: list[str]) -> dict[str, float]`
       - For each signal_key, compute overall Brier using all entries.
       - Uses `blender.compute_rolling_brier(entries, signal_key, window=len(entries))` effectively — all entries up to now.
       - Returns dict {signal_key: brier}.

    ```python
    def _run_governance(
        entries: list[dict],   # prediction_history entries
        versions: dict,        # current version state
        signal_keys: list[str],
        blend_weights: dict[str, float],
        startup: bool = False,
    ) -> dict:
        """Run one governance cycle: compute metrics, check drift, build snapshot.
        
        Args:
            entries: Deduplicated prediction_history entries.
            versions: Current version state from load_versions().
            signal_keys: Ordered list of active signal keys (e.g. ["elo", "market_odds", "catboost", "form", "lineup_strength"]).
            blend_weights: Current blend weights from calibrate_and_blend().
            startup: True if called during startup (triggers backtest + reference baseline init).
        
        Returns:
            dict: Run snapshot per D-06 schema.
        """
    ```

    Implementation:
    1. Deduplicate entries via `_deduplicate_history()`
    2. Compute per-signal rolling Brier via `blender.compute_rolling_brier(entries, key, window=50)` for each signal_key.
    3. Compute reference baselines if not yet set (store in versions dict).
    4. For each signal_key, call `check_drift(entries, key, reference_baseline, window=50, sigma_threshold=2.0)`.

       - blended_brier: per-signal briers via blender compute_rolling_brier for key "blended" if available
