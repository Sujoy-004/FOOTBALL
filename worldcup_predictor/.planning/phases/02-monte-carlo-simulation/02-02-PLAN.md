  <action>
    Create `scripts/benchmark_simulation.py` (creates `scripts/` directory which does not exist yet — use `mkdir`).

    **Script requirements (per D-11, D-12):**

    1. **Shebang:** `#!/usr/bin/env python3`
    2. **Argparse:** `--iterations` flag, default 50000, type int. On `--help`, shows description.
    3. **Data loading:** Call `state.load_teams(DATA_DIR)`, `state.load_bracket(DATA_DIR)`, `state.load_played(DATA_DIR)`. Import `DATA_DIR` from `src.constants`. Run from project root (cwd = worldcup_predictor/).
    4. **Warmup:** Call `run_simulation(teams, bracket, played, iterations=100)` before timing. This triggers JIT compilation, CPU cache warmup, and module-level initializations.
    5. **Timing:** 
       - `start = time.perf_counter()`
       - `probs = run_simulation(teams, bracket, played, iterations=args.iterations)`
       - `elapsed = time.perf_counter() - start`
    6. **Output format:**
       ```
       Monte Carlo Simulation Benchmark
       ────────────────────────────────
       Iterations:    50000
       Elapsed:       1.234s
       Rate:          40518 sims/sec
       Threshold:     5.000s
       Status:        PASS
       ────────────────────────────────
       Champion sum:  1.0000
       ```
       - Columns: left-aligned label (15 chars), right-aligned value
       - `elapsed` formatted as `{elapsed:.3f}s`
       - `rate` formatted as `{rate:.0f} sims/sec`
       - `threshold` always `5.000s`
       - `status`: `PASS` if `elapsed < 5.0`, else `FAIL`
    7. **Sanity check:** After timing, compute `sum(p["champion"] for p in probs.values())`, format as `{champion_sum:.4f}`. Print `"Champion sum:"` aligned.
    8. **Return code:** Exit 0 on PASS, exit 1 on FAIL (for scripting purposes).

    Follow the existing script patterns in the project — no classes, procedural `main()` guarded by `if __name__ == "__main__"`.

    **IMPORTANT path resolution:** The script will be run as `python scripts/benchmark_simulation.py` from the `worldcup_predictor/` directory. Use the same path resolution pattern: import from `src.*` works because `worldcup_predictor/` is the working directory. No `sys.path` manipulation needed.

    After creation, verify the script runs successfully:
    ```bash
    cd worldcup_predictor && python scripts/benchmark_simulation.py --iterations 100
    ```
    This should run with 100 iterations (fast) and print the output format.
