  <name>Task 1: Create src/predictors/form.py — Form signal computation</name>
  <files>src/predictors/form.py</files>
  <action>
    Create a new module at `src/predictors/form.py` implementing the Elo-based
    form residual signal. Per D-01 through D-07.

    **Form computation formula:**
    - Per-team residual = average of (actual - expected_score) over last N matches
    - match-level: `form_delta = home_residual - away_residual`
    - match-level probability: `p_form = sigmoid(k * form_delta)`
    - Where `expected_score(elo_a, elo_b) = 1 / (1 + 10 ^ ((elo_b - elo_a) / 400))`

    **Algorithm:**
    1. Collect ALL match results from played_groups.json + played.json (D-03)
    2. For each match, determine teams, who won (1.0/0.5/0.0), and timestamp
    3. Compute expected_score for each match using current Elo from teams dict
    4. Build per-team chronologically sorted list of residuals
    5. For each team, take last FORM_WINDOW_SIZE residuals, compute mean
    6. If a team has 0 matches, residual = None (signal unavailable for that team)

    **Module structure:**

    ```python
    """Form signal — rolling Elo residual over last N matches.

    Computes per-team form as average of (actual - expected_score) over a
    configurable rolling window of recent matches. If either team in a match
    has 0 matches played, the form signal returns available: false for that match.

    Pure Python stdlib (math.log, math.exp). No numpy, no sklearn (D-01 from Phase 14).
    """

    import logging
    import math
    from datetime import datetime, timezone
    from typing import Any

    from src import constants
    from src.elo import expected_score

    logger = logging.getLogger(__name__)


    def _actual_from_match(match: dict) -> float:
        """Convert match outcome to actual score (1.0 win, 0.5 draw, 0.0 loss)."""
        winner = match.get("winner")
        team_a = match.get("team_a")
        if winner == team_a:
            return 1.0
        if winner is None:
            return 0.5
        return 0.0  # away won


    def _collect_team_residuals(
        played: dict[str, dict],
        played_groups: dict[str, dict],
        teams: dict[str, dict],
    ) -> dict[str, list[tuple[str, float]]]:
        """Build per-team chronologically sorted list of (match_id, residual) pairs.

        Iterates all matches from both played and played_groups, computes residuals
        as (actual - expected_score), and appends to each team's list.
        Returns dict: team_name -> [(match_id, residual), ...] sorted by completed_at.
        """
        residuals: dict[str, list[tuple[str, float]]] = {}

        def _process_match(mid: str, m: dict) -> None:
            ta = m.get("team_a", "")
            tb = m.get("team_b", "")
            if not ta or not tb:
                return
            if ta not in teams or tb not in teams:
                return
            elo_a = teams[ta]["elo"]
            elo_b = teams[tb]["elo"]
            exp_a = expected_score(elo_a, elo_b)
            actual = _actual_from_match(m)
            residual_a = actual - exp_a
            residual_b = (1.0 - actual) - (1.0 - exp_a)  # = exp_a - actual = -residual_a
            timestamp = m.get("completed_at", "")

            for team, res in [(ta, residual_a), (tb, residual_b)]:
                if team not in residuals:
                    residuals[team] = []
                residuals[team].append((mid, res, timestamp))

        for mid, m in played.items():
            _process_match(mid, m)
        for mid, m in played_groups.items():
            _process_match(mid, m)

        # Sort each team's residuals by completed_at, then match_id
        for team in residuals:
            residuals[team].sort(key=lambda x: (x[2], x[0]))

        return residuals


    def _compute_form_residuals(
        team_residuals: dict[str, list[tuple[str, float, str]]],
        window_size: int = 5,
    ) -> dict[str, float | None]:
        """Compute per-team form residual as average of last N residuals.

        If a team has 0 matches, residual is None (unavailable).
        If a team has fewer than window_size matches, uses all available.
        """
        result: dict[str, float | None] = {}
        for team, entries in team_residuals.items():
            if not entries:
                result[team] = None
            else:
                recent = entries[-window_size:]
                avg = sum(r[1] for r in recent) / len(recent)
                result[team] = avg
        return result


    def compute_form_signal(
        teams: dict[str, dict],
        groups: dict,
        bracket: list[dict] | None = None,
        played: dict[str, dict] | None = None,
        played_groups: dict[str, dict] | None = None,
        window_size: int | None = None,
        k: float | None = None,
    ) -> dict:
        """Compute form signal for all matches in groups and bracket.

        Args:
            teams: Teams dict with {team_name: {elo: float}}.
            groups: Groups dict (for group match iteration).
            bracket: Optional bracket list (for knockout match iteration).
            played: Played knockout matches dict or None.
            played_groups: Played group matches dict or None.
            window_size: Rolling window size (default: FORM_WINDOW_SIZE from constants).
            k: Sigmoid steepness (default: DEFAULT_FORM_K from constants).

        Returns:
            Cache dict with keys: fetched_at, expires_at, matches.
            matches: {match_id: {probability: float|None, available: bool,
                                 timestamp: str, reason: str|None}}.
        """
        if window_size is None:
            window_size = constants.FORM_WINDOW_SIZE
        if k is None:
            k = constants.DEFAULT_FORM_K

        played = played or {}
        played_groups = played_groups or {}
        now = datetime.now(timezone.utc)

        # Compute per-team form residuals
        team_residuals = _collect_team_residuals(played, played_groups, teams)
        residuals = _compute_form_residuals(team_residuals, window_size)

        matches: dict[str, dict] = {}
        groups_data = groups.get("groups", groups) if groups else {}

        # Process group matches
        for letter in groups_data:
            grp = groups_data[letter]
            for match_def in grp.get("matches", []):
                mid = match_def["match_id"]
                home = match_def["team_a"]
                away = match_def["team_b"]
                res_home = residuals.get(home)
                res_away = residuals.get(away)
                if res_home is None or res_away is None:
                    unavail_teams = []
                    if res_home is None:
                        unavail_teams.append(home)
                    if res_away is None:
                        unavail_teams.append(away)
                    matches[mid] = {
                        "probability": None,
                        "timestamp": now.isoformat(),
                        "available": False,
                        "reason": f"no_matches_for_{'_'.join(unavail_teams)}",
                    }
                else:
                    form_delta = res_home - res_away
                    raw_p = 1.0 / (1.0 + math.exp(-k * form_delta))
                    # Clamp to [1e-15, 1-1e-15]
                    raw_p = max(1e-15, min(1 - 1e-15, raw_p))
                    matches[mid] = {
                        "probability": round(raw_p, 6),
                        "timestamp": now.isoformat(),
                        "available": True,
                    }

        # Process knockout bracket matches
        if bracket:
            for match_def in bracket:
                mid = match_def["match_id"]
                home = match_def.get("team_a")
                away = match_def.get("team_b")
                if not home or not away:
                    continue  # slot not yet resolved (dependent on prior round)
                res_home = residuals.get(home)
                res_away = residuals.get(away)
                if res_home is None or res_away is None:
                    unavail_teams = []
                    if res_home is None:
                        unavail_teams.append(home)
                    if res_away is None:
                        unavail_teams.append(away)
                    matches[mid] = {
                        "probability": None,
                        "timestamp": now.isoformat(),
                        "available": False,
                        "reason": f"no_matches_for_{'_'.join(unavail_teams)}",
                    }
                else:
                    form_delta = res_home - res_away
                    raw_p = 1.0 / (1.0 + math.exp(-k * form_delta))
                    raw_p = max(1e-15, min(1 - 1e-15, raw_p))
                    matches[mid] = {
                        "probability": round(raw_p, 6),
                        "timestamp": now.isoformat(),
                        "available": True,
                    }

        # Phase 14a: upsert into permanent prediction ledger
        if matches:
            try:
                from src.state import ledger_upsert
                for mid, entry in matches.items():
                    ledger_upsert(mid, "form", entry)
            except Exception:
                logger.warning("Failed to upsert form signal into prediction ledger", exc_info=True)

        return {
            "fetched_at": now.isoformat(),
            "expires_at": (now + timedelta(hours=1)).isoformat(),  # 1h TTL — recomputed fresh every cycle
            "matches": matches,
        }
    ```

    - Uses `expected_score()` from src.elo for Elo-based residual computation (D-01)
    - Uses teams.json CURRENT Elo values (approximation; replaying full Elo history for
      exact match-time ratings is out of scope)
    - Empty/unavailable teams produce `available: false` with reason string (D-16)
    - Uses `math.exp` (stdlib) for sigmoid — NOT numpy (D-01 from Phase 14)
    - Graceful degradation: if match resolution fails, skip that match entry
      (never crash)
    - Edge case: if both teams have 0 matches, the match gets available:false (D-02)

  <name>Task 2: Create src/predictors/lineup.py — Lineup strength signal</name>
  <files>src/predictors/lineup.py</files>
  <action>
    Create a new module at `src/predictors/lineup.py` implementing the lineup
    strength signal from squad market values. Per D-08 through D-14.

    **Lineup computation formula:**
    - `strength_delta = ln(home_value / away_value)` (D-08, D-09, D-11)
    - `p_strength = sigmoid(k * strength_delta)`

    **Algorithm:**
    1. Load team_values.json via state.load_team_values()
    2. For each match in groups and bracket, look up home and away values
    3. If either team missing from team_values, signal unavailable for that match
    4. Compute strength_delta and apply sigmoid

    **Module structure:**

    ```python
    """Lineup strength signal — squad market value proxy.

    Computes match-level win probability from aggregate squad market values
    using log-ratio formulation: p = sigmoid(k * ln(home_value / away_value)).

    Data source: static data/team_values.json (aggregate squad market values in EUR).
    NOT from BSD API per user decision (expired API key, 832 API calls, live dependency).

    Pure Python stdlib (math.log, math.exp). No numpy, no sklearn (D-01 from Phase 14).
    """

    import logging
    import math
    from datetime import datetime, timedelta, timezone

    from src import constants
    from src.state import load_team_values

    logger = logging.getLogger(__name__)


    def compute_lineup_signal(
        groups: dict,
        bracket: list[dict] | None = None,
        team_values: dict[str, int] | None = None,
        k: float | None = None,
    ) -> dict:
        """Compute lineup strength signal for all matches in groups and bracket.

        Args:
            groups: Groups dict (for group match iteration).
            bracket: Optional bracket list (for knockout match iteration).
            team_values: Pre-loaded team values dict. If None, loads from disk.
            k: Sigmoid steepness (default: DEFAULT_LINEUP_K from constants).

        Returns:
            Cache dict with keys: fetched_at, expires_at, matches.
            matches: {match_id: {probability: float|None, available: bool,
                                 timestamp: str, reason: str|None}}.
        """
        if k is None:
            k = constants.DEFAULT_LINEUP_K
        if team_values is None:
            team_values = load_team_values()

        now = datetime.now(timezone.utc)
        matches: dict[str, dict] = {}
        groups_data = groups.get("groups", groups) if groups else {}

        def _compute_match(mid: str, home: str, away: str) -> None:
            if home not in team_values or away not in team_values:
                missing = []
                if home not in team_values:
                    missing.append(home)
                if away not in team_values:
                    missing.append(away)
                matches[mid] = {
                    "probability": None,
                    "timestamp": now.isoformat(),
                    "available": False,
                    "reason": f"no_value_for_{'_'.join(missing)}",
                }
                return

            home_val = team_values[home]
            away_val = team_values[away]
            if home_val <= 0 or away_val <= 0:
                matches[mid] = {
                    "probability": None,
                    "timestamp": now.isoformat(),
                    "available": False,
                    "reason": "non_positive_value",
                }
                return

            ratio = home_val / away_val
            strength_delta = math.log(ratio)
            raw_p = 1.0 / (1.0 + math.exp(-k * strength_delta))
            raw_p = max(1e-15, min(1 - 1e-15, raw_p))
            matches[mid] = {
                "probability": round(raw_p, 6),
                "timestamp": now.isoformat(),
                "available": True,
            }

        # Process group matches
        for letter in groups_data:
            grp = groups_data[letter]
            for match_def in grp.get("matches", []):
                _compute_match(match_def["match_id"], match_def["team_a"], match_def["team_b"])

        # Process knockout bracket matches
        if bracket:
            for match_def in bracket:
                mid = match_def["match_id"]
                home = match_def.get("team_a")
                away = match_def.get("team_b")
                if not home or not away:
                    continue  # slot not yet resolved
                _compute_match(mid, home, away)

        # Phase 14a: upsert into permanent prediction ledger
        if matches:
            try:
                from src.state import ledger_upsert
                for mid, entry in matches.items():
                    ledger_upsert(mid, "lineup_strength", entry)
            except Exception:
                logger.warning("Failed to upsert lineup signal into prediction ledger", exc_info=True)

        return {
            "fetched_at": now.isoformat(),
            "expires_at": (now + timedelta(hours=1)).isoformat(),
            "matches": matches,
        }
    ```

    Key implementation notes:
    - `team_values` parameter allows pre-loading (test-friendly) or auto-loads from disk
    - `math.log` for natural log (stdlib only — D-08)
    - `math.exp` for sigmoid (stdlib only)
    - Missing teams → available:false with descriptive reason (D-16)
    - Zero/negative values → available:false (data integrity check)
    - Both group matches AND bracket matches computed (bracket slots with unresolved
      teams skipped gracefully)
    - Upserts to prediction ledger (D-15) — same pattern as odds.py and catboost.py
    </action>
